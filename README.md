### Setup

Create a `.env` file with your API credentials:

```env
API_BASE_URL=https://assessment.ksensetech.com/api
API_KEY=your_api_key_here
```

The `.env` file is listed in `.gitignore` so it won't be committed. The `.env.example` file shows the required format without exposing actual secrets.
